# Crops Performance Analytics
## Project Overview

This project analyzes crop performance across different regions in Kenya using Microsoft Excel and Power BI.

The objective is to provide actionable insights into crop production trends, regional performance, and overall agricultural productivity.

--- 

## Objectives

- Analysing crop performance accross regions
- Compare crop performance
- Identify top-performing regions
- Track productivity trends
- Build an interactive Power BI dashboard

--- 

  ## Tools used

  - Microsoft Excel
  - Microsoft Power Bi
  - Github
